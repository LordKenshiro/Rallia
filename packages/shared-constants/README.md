# Shared Constants Package

Colors, animation timings, and other constants used across Rallia apps.

## Usage

```typescript
import { COLORS, ANIMATION_DELAYS } from '@rallia/shared-constants';
```

## Available Constants

### COLORS

- Primary colors (turquoise, blue, orange, red, green)
- Background colors
- Text colors
- Border colors

### ANIMATION_DELAYS

- Animation timing constants
