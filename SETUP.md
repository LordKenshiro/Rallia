## Prerequisites
- Node.js v18+
- npm OR yarn
- Expo CLI (`npm install -g expo-cli`)

## 1. Supabase Backend
- Go to [https://supabase.com](https://supabase.com), sign in/register.
- Get your `SUPABASE_URL` and `SUPABASE_ANON_KEY` from Project API Settings.

## 2. Launch app
- cd rallia-app
- npm start
