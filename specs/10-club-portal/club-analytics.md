# Club Analytics

## Overview

Performance metrics and insights for club managers.

## Dashboard

### Key Metrics

| Metric            | Description                 |
| ----------------- | --------------------------- |
| Total Bookings    | All-time booking count      |
| This Period       | Bookings in selected period |
| Utilization Rate  | % of available slots booked |
| Cancellation Rate | % of bookings cancelled     |
| No-Show Rate      | % of no-shows               |
| Revenue           | Total and period revenue    |

## Visualizations

### Bookings Over Time

Line chart showing:

- Daily/weekly/monthly bookings
- Compare to previous period

### Utilization Heatmap

Grid showing:

- Rows: Time slots
- Columns: Days of week
- Color: Utilization %

Helps identify:

- Peak hours
- Underutilized slots

### Revenue Breakdown

- By court
- By sport (Tennis vs Pickleball)
- By time of day
- By day of week

## Filters

### Time Period

- Today
- This week
- This month
- This year
- Custom range

### Sport

- All
- Tennis only
- Pickleball only

## Reports

### Bookings Report

| Column  | Data                        |
| ------- | --------------------------- |
| Date    | Booking date                |
| Time    | Time slot                   |
| Court   | Court name                  |
| Player  | Player name                 |
| Status  | Completed/Cancelled/No-show |
| Revenue | Amount                      |

Export as CSV for accounting.

### Utilization Report

| Column    | Data                  |
| --------- | --------------------- |
| Time Slot | Hour                  |
| Available | Total available slots |
| Booked    | Slots booked          |
| Rate      | % utilization         |
