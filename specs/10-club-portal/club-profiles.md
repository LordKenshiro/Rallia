# Club Profiles

## Club Registration

### Sign-Up Methods

- Email + Password
- Facebook, Google, Apple ID

### Verification

Consider verification process:

- Business email
- Phone verification
- Manual approval by Rallia team

## Club Profile Information

| Field         | Required | Description                 |
| ------------- | -------- | --------------------------- |
| Club Name     | Yes      | Official name               |
| Address       | Yes      | Full street address         |
| Phone         | Yes      | Contact number              |
| Email         | Yes      | Contact email               |
| Logo          | No       | Club logo image             |
| Photos        | No       | Facility photos             |
| Court Types   | Yes      | Tennis, Pickleball, or both |
| Surface Types | Yes      | Clay, Hard, Grass, Indoor   |
| Opening Hours | Yes      | Weekly schedule             |
| Description   | No       | About the club              |

## Court Information

For each court:

| Field             | Description                   |
| ----------------- | ----------------------------- |
| Court Name/Number | e.g., "Court 1", "Main Court" |
| Sport             | Tennis or Pickleball          |
| Surface           | Clay, Hard, etc.              |
| Indoor/Outdoor    | Covered status                |
| Features          | Lighting, seating, etc.       |
