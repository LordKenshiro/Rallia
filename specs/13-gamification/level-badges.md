# Level Badges

## Certified Level Badge

### Criteria

Player's skill level has been verified through:

- Peer references (3 certified players at same level or higher), OR
- Proof submission (videos, external ratings, tournament results)

### Display

| Location       | Appearance        |
| -------------- | ----------------- |
| Next to level  | ✓ checkmark       |
| Profile        | "Certified" label |
| Search results | Checkmark icon    |

### Example

```
Jean Dupont
NTRP 4.0 ✓
```

### Earning the Badge

See [Level Certification](../04-player-rating/level-certification.md).

### Losing the Badge

- Requesting a level increase → loses badge, must re-certify
- Automatic level change → keeps badge at new level
- Requesting a level decrease → keeps badge at new level

## Badge Implications

### Certified Players Can:

- Give references to other players (post-match or on request)
- Have higher visibility in search results (optional filter)

### Certified Matches

Matches between certified players have:

- Higher trust level
- Post-match references can be given by both players

## Future Badges

Consider adding:

- **Verified Pro**: Tournament-level or professional player
- **Rising Star**: Player who improved significantly
- **Consistent**: Level stable over many matches
