# Reputation Badges

## High Reputation Badge

### Criteria

Reputation score ≥ 90%

### Display

| Location       | Appearance                             |
| -------------- | -------------------------------------- |
| Profile        | ⭐ star icon + "Excellent Reliability" |
| Search results | Star indicator                         |
| Match cards    | Star icon                              |

### Example

```
Jean Dupont
Reputation: 94% ⭐
Excellent Reliability
```

### Earning the Badge

1. Complete at least 3 matches (reputation becomes active)
2. Maintain reputation at or above 90%
3. Badge awarded automatically

### Losing the Badge

When reputation drops below 90%:

- Badge removed immediately
- Notification sent with encouragement
- Can be re-earned by improving reputation

### Impact

High reputation badge holders:

- More visible in search (optional filter)
- Eligible for Most Wanted Player super-badge
- Inspire trust in potential opponents
