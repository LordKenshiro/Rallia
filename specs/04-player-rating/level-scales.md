# Level Scales

## Tennis: NTRP Scale

The **National Tennis Rating Program (NTRP)** is the standard used by USTA and Tennis Canada.

### Scale Range: 1.0 - 7.0

| Level | Description                                                               |
| ----- | ------------------------------------------------------------------------- |
| 1.0   | Just starting to play tennis                                              |
| 1.5   | Limited experience, working on getting the ball in play                   |
| 2.0   | Needs on-court experience, has obvious stroke weaknesses                  |
| 2.5   | Learning to judge where the ball is going, weak court coverage            |
| 3.0   | Fairly consistent on medium-paced shots, not comfortable with all strokes |
| 3.5   | Achieved improved stroke consistency, developing variety                  |
| 4.0   | Dependable strokes with directional control and depth                     |
| 4.5   | Beginning to master variety and power, good court coverage                |
| 5.0   | Good shot anticipation, frequently hits winners                           |
| 5.5   | Power and consistency as major weapons                                    |
| 6.0   | Trained intensively for national/international competition                |
| 6.5   | Extensive satellite or challenger tournament experience                   |
| 7.0   | World-class professional player                                           |

### Certification Threshold

Players at **NTRP 3.0 and above** are encouraged to get certified through peer references.

## Pickleball: DUPR Scale

The **Dynamic Universal Pickleball Rating (DUPR)** is the standard for pickleball.

### Scale Range: 2.0 - 8.0

| Level     | Description                                             |
| --------- | ------------------------------------------------------- |
| 2.0 - 2.5 | Beginner, learning basic rules and strokes              |
| 3.0 - 3.5 | Developing player, can sustain short rallies            |
| 4.0 - 4.5 | Intermediate, consistent strokes and strategy awareness |
| 5.0 - 5.5 | Advanced, strong court positioning and shot selection   |
| 6.0 - 6.5 | Expert, tournament-level play                           |
| 7.0+      | Professional/Elite level                                |

### Certification Threshold

Players at **DUPR 3.5 and above** are encouraged to get certified through peer references.
