# User Metrics

## Overview

Metrics related to player accounts and behavior.

## Player Counts

### Total Players

| Metric         | Description               | Breakdown                      |
| -------------- | ------------------------- | ------------------------------ |
| Total Players  | All-time registered users | By sport, by region            |
| New Players    | Signups in period         | Daily, weekly, monthly, yearly |
| Active Players | Logged in within period   | DAU, WAU, MAU                  |

### Retention

| Metric        | Description                         |
| ------------- | ----------------------------------- |
| D1 Retention  | % users active 1 day after signup   |
| D7 Retention  | % users active 7 days after signup  |
| D30 Retention | % users active 30 days after signup |
| Churn Rate    | % users who stopped using app       |
