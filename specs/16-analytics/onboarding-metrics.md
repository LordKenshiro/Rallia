# Onboarding Metrics

## Overview

Metrics for understanding the signup and onboarding experience.

## Funnel Metrics

### Onboarding Funnel

```
App Open → Sign-In → Pre-Onboarding → Onboarding → Complete
```

| Stage               | Users | Conversion | Drop-off |
| ------------------- | ----- | ---------- | -------- |
| App Opened          | 1000  | -          | -        |
| Sign-In Complete    | 850   | 87%        | 13%      |
| Sport Selected      | 800   | 94%        | 6%       |
| Onboarding Complete | 650   | 81%        | 19%      |

### Completion Rate

| Metric                  | Value                      |
| ----------------------- | -------------------------- |
| Overall Completion Rate | % who finish onboarding    |
| Partial Completion      | Stopped at specific screen |
| Comeback Rate           | Started later, completed   |
