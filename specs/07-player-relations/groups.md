# Groups

## Overview

Small private groups of players who regularly play together. Maximum 10 members.

## Creating a Group

1. Tap "Create Group"
2. Name the group
3. Add members (up to 10)
4. Group created
5. Group chat automatically created

## Group Properties

| Property   | Value                     |
| ---------- | ------------------------- |
| Max Size   | 10 players                |
| Visibility | Private (members only)    |
| Creator    | Automatically a moderator |
| Chat       | Automatically created     |

## Roles

### Moderator

- Can add members
- Can remove members
- Can assign other moderators
- Can delete the group
- Creator is moderator by default

### Member (Non-Moderator)

- Can add members
- Cannot remove members
- Can leave the group
- Can use group chat
- Can receive match invitations

## Actions

### Adding Members

| Who        | Can Add |
| ---------- | ------- |
| Moderators | ✅ Yes  |
| Members    | ✅ Yes  |

> **Note:** Following WhatsApp pattern, members can be added without their prior approval, but they can leave at any time.

### Removing Members

| Who        | Can Remove |
| ---------- | ---------- |
| Moderators | ✅ Yes     |
| Members    | ❌ No      |

### Leaving the Group

Any member can leave at any time:

1. Tap "Leave Group"
2. Confirm
3. Removed from group and chat

If last moderator leaves:

- Next most senior member becomes moderator
- If no members left, group is deleted

## Group Chat

Automatically created with the group. See [Communications](../08-communications/README.md).

## Using Groups for Matches

When creating a match:

1. Select "Private Match"
2. Choose a group as recipient
3. All group members receive invitation
4. First to accept gets the spot (or manual approval)

## Sport Separation

Groups are sport-specific:

- Tennis groups
- Pickleball groups

Cannot have cross-sport groups.
