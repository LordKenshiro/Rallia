# Visual Differentiation

## Overview

Each sport universe has distinct visual identity to prevent confusion.

## Color Schemes

### Tennis Universe

| Element | Color Suggestion            |
| ------- | --------------------------- |
| Primary | Blue tones                  |
| Accent  | Traditional tennis green    |
| Icons   | Tennis-specific iconography |

### Pickleball Universe

| Element | Color Suggestion                |
| ------- | ------------------------------- |
| Primary | Green tones                     |
| Accent  | Pickleball-specific colors      |
| Icons   | Pickleball-specific iconography |

## Visual Elements

### Always Visible Indicators

1. **Header/Navigation Bar**
   - Sport-specific color
   - Sport name or icon visible

2. **Tab Bar / Bottom Navigation**
   - Sport-specific accent colors
   - Optional sport indicator

3. **App Icon (optional)**
   - Consider adaptive icon showing current sport mode

### Sport-Specific Assets

| Asset Type          | Tennis        | Pickleball        |
| ------------------- | ------------- | ----------------- |
| Ball icon           | Tennis ball   | Pickleball        |
| Racket icon         | Tennis racket | Pickleball paddle |
| Court illustration  | Tennis court  | Pickleball court  |
| Background patterns | Tennis-themed | Pickleball-themed |
